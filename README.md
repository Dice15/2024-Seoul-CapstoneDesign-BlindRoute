### Works
- GPT Assistants를 활용하여 시각장애인 버스 안내 기능을 제공하는 챗봇 구현
- PWA 기술을 적용하여 Android 스마트폰에서 앱으로 설치할 수 있도록 함
- 챗봇과의 대화, 경로 찾기, 실시간 버스 도착 시간 REST API 구축
- 웹 앱 UI 디자인 및 구현
